üîç GET /books/<int:book_id>

> Description: Get details of a single book by ID.

> Parameters:

> book_id (int): ID of the book

> Response:

> 200 OK:

    {
        "id": 1,
        "title": "Two States",
        "author": "Chetan Bhagat"
    }

> 404 Not Found:

    "Book not found"


‚ûï POST /books

> Description: Add a new book.

> Request Body:

    {
         "title": "Wings of Fire",
         "author": "A.P.J. Abdul Kalam"
    }
> Response:

    201 Created

    {
      "id": 3,
      "title": "Wings of Fire",
      "author": "A.P.J. Abdul Kalam"
    }

‚úèÔ∏è PUT /books/<int:book_id>

> Description: Update an existing book.

> Parameters:

    book_id (int): ID of the book to update

> Request Body (partial or full):

    {
        "title": "Updated Title",
        "author": "Updated Author"
    }

> Response:

    200 OK

    {
        "id": 1,
        "title": "Updated Title",
        "author": "Updated Author"
    }

> 404 Not Found:

  "Not found"



‚ùå DELETE /books/<int:book_id>

> Description: Delete a book by ID.

> Parameters:

    book_id (int): ID of the book

> Response:

   204 No Content
